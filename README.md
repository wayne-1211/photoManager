# 📸 Photo Manager

**Photo Manager** 是一款專為攝影師、活動紀錄者與展會工作人員打造的「純前端」極速照片篩選與分類工具。

不需要架設任何後端伺服器，也不用將私人照片上傳到雲端。只需用瀏覽器打開網頁，即可在本地端極速載入大量高畫質原圖，並透過全鍵盤操作完成挑片、EXIF 分析與大檔分割打包，大幅提升活動結束後的照片整理效率。

---

## ✨ 核心特色

*   **🔒 100% 離線本地運行：** 純 HTML/CSS/JS 開發，照片處理與打包完全在你的瀏覽器內完成，零隱私外洩風險。
*   **⚡ 極速預覽體驗：** 獨家「EXIF 縮圖提取」與「非同步分塊載入」技術，即使一次匯入數百張單眼相機高解析度原圖（如 Sony a6700），也不會卡死瀏覽器。
*   **⌨️ 全鍵盤流暢挑片：** 專為高效率設計的 Master-Detail 雙欄佈局。左手數字鍵分類、右手方向鍵切換，配合大圖背景預載入機制，實現無縫挑片。
*   **📊 EXIF 專業統計：** 自動解析並統計焦段（10mm 級距）、光圈、快門、ISO 與相機型號，甚至支援讀取 Sony 專屬的「創意風格 (Creative Style)」。
*   **📦 智慧大檔分割匯出：** 依據自訂分類自動建立資料夾結構。當匯出檔案超過 500MB 時，會自動採用標準 ZIP 分割模式（`.zip.001`, `.zip.002`）下載，避免瀏覽器記憶體崩潰（OOM）。
*   **📱 支援 PWA 安裝：** 可直接將網頁安裝為桌面應用程式，享受無邊框的沉浸式獨立軟體體驗。

## 🚀 快速開始

### 選項一：直接使用（瀏覽器執行）
1. 將專案資料夾下載至本地電腦。
2. 雙擊打開 `index.html` 即可開始使用。

### 選項二：安裝為桌面軟體 (PWA)
1. 將專案部署至 GitHub Pages 或任何靜態網頁伺服器。
2. 使用 Chrome 或 Edge 瀏覽器開啟網址。
3. 點擊網址列右側的「安裝應用程式」圖示，即可將 Photo Manager 加到電腦桌面。

---

## 🎮 快捷鍵與操作指南

在 **「照片管理」** 分頁中，你可以完全捨棄滑鼠，享受極速分類體驗：

| 快捷鍵 | 動作 | 說明 |
| :--- | :--- | :--- |
| `↑` `↓` `←` `→` | **移動焦點** | 在右側縮圖網格中自由切換當前選取的照片。 |
| `1` ~ `4` | **快速標記分類** | 將當前照片標記為對應的分類（如：機器人特寫、賽場動態等），並**自動跳轉至下一張**。 |
| `5` | **廢片標記** | 預設為不匯出，標記後自動跳轉至下一張。 |

> **💡 提示：** 分類名稱可以在介面上方自由修改，也可以勾選是否要將該分類加入最終的 ZIP 打包清單中。

## 🛠 技術架構

*   **前端核心：** 原生 HTML5, CSS3 (CSS Grid/Flexbox), Vanilla JavaScript (ES6+)。
*   **EXIF 解析：** 客製化二進位 DataView 讀取，精準提取 IFD0, ExifIFD 與 IFD1 內建縮圖，支援自動修正直拍照片的 `Orientation` 視覺溢出問題。
*   **套件依賴：** 
    *   [JSZip](https://stuk.github.io/jszip/) - 用於純前端建立 ZIP 壓縮檔。
    *   [FileSaver.js](https://github.com/eligrey/FileSaver.js/) - 用於觸發瀏覽器下載。

---

## 📝 注意事項

*   **支援格式：** 目前主要支援 `.jpg` / `.jpeg` 檔案。RAW 檔因各家廠商編碼不同，純前端無法直接解析。
*   **效能建議：** 雖然軟體具備防當機機制，但仍受限於瀏覽器分配的實體記憶體限制。建議單次處理總量控制在 1000 張以內，以獲得最佳流暢度。
*   **多卷解壓縮：** 若下載了分割壓縮檔（`.zip.001`, `.zip.002`...），請將它們放在同一個資料夾內，並使用 **7-Zip** 等解壓縮軟體對準第一個檔案進行「合併/解壓縮」即可。
